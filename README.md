##Real-time-Emotion-Recognition-with-TTS

Real-time Emotion Recognition with TTS

Detects your facial emotions in real-time using **DeepFace** and **OpenCV**, and speaks them using **pyttsx3**.



Features
- Real-time emotion detection using DeepFace.
- Speaks the detected emotion loudly.
- Detects multiple emotions as you change expressions.
- Displays emotion on the video feed.
- Close program by pressing `'q'` or clicking the window close button.
